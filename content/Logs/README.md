## Structure

```
Logs/
├── Daily/          — Daily check-ins (9 AM)
│   └── 2026-04-25.md
├── Weekly/         — Weekly reviews (Saturday evening)
│   └── 2026-W17.md
└── README.md       — This file
```

---

## Daily Check-ins

**When:** Every day at 9:00 AM
**Purpose:** Set objectives, align with goals, ADD accountability
**Format:**
- Objectives for the day
- Alignment with overall goals
- Distraction risks
- End-of-day follow-up (if provided)

## Weekly Reviews

**When:** Every Saturday evening
**Purpose:** Reflect on the week, track progress, adjust plans
**Format:**
- What was accomplished this week
- What was planned but not done (and why)
- Goal alignment score
- Key wins and blockers
- Plan for next week
- ADD-specific reflections

---

*All entries are logged automatically by Koko and synced via iCloud.*
