Context is not memory. Context is a weapon.

> Context is the minimum knowledge an agent needs to be useful here.

Not the raw dump from Layer 1. Not the problem frame from Layer 2. Context answers one question: what does something reading this folder absolutely need to know?

The layer contains five living files:

- `glossary.md` — terms defined once, referenced everywhere
- `facts.md` — claims that have been verified, with sources
- `decisions.md` — why X won over Y, timestamped
- `patterns.md` — insights that keep recurring across iterations
- `summary.md` — the current state of the workspace, updated automatically

This layer is maintained by the AI and reviewed by the human. The agent updates `facts.md` after every research pass. It appends to `decisions.md` when you choose a direction. The human checks for hallucinations. The human corrects definitions.

> Storage hoards. Context prepares.

The difference is outcome. An agent reading a hundred raw files produces noise. An agent reading five curated context files produces signal. This layer is the signal.
